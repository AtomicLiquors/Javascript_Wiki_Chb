# 환경설정
- 모든 브라우저 + Node.js는 자바스크립트 엔진을 내장하고 있다. 자바스크립트의 중추인 ECMAScript를 실행할 수 있다.



하지만 ECMAScript 이외의 기능은 호환되지 않는다.
- DOM API : 파싱된 html 요소를 조작하는 기능의 집합 는 브라우저만 제공한다.

- 파일 수정, 생성이 가능한 파일 시스템은
Node.js만 제공한다.
